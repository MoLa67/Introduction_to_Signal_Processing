# Introduction to Signal Processing Laboratory

This repository contains a series of self-guided **Google Colab** laboratory notebooks developed for an undergraduate **Introduction to Signal Processing** module. The labs combine theoretical explanations, guided programming exercises, and practical experiments using **Python**, **NumPy**, **SciPy**, **Matplotlib**, and **SymPy**.

The laboratory sessions are designed to reinforce lecture material by enabling students to explore signal processing concepts through simulation, visualization, and computational analysis. Each practical includes a brief theoretical background, clearly defined exercises, and an accompanying solution notebook for instructors.

## Laboratory 1 – Signals and Systems Fundamentals

This laboratory introduces the Python programming environment for signal processing and covers the fundamental concepts of continuous-time and discrete-time signals. Students learn to generate and visualize signals, investigate signal properties, and perform basic signal operations including addition, multiplication, shifting, and time reversal.

**Topics**

* Python programming for signal processing
* Signal visualization
* Continuous-time and discrete-time signals
* Periodic, aperiodic, even and odd signals
* Signal operations

---

## Laboratory 2 – Linear Time-Invariant Systems

This laboratory focuses on the analysis of Linear Time-Invariant (LTI) systems using Python. Students investigate discrete-time and continuous-time convolution, impulse and step responses, and frequency response analysis. Practical exercises demonstrate the relationship between theoretical system analysis and computational implementation.

**Topics**

* Discrete-time convolution
* Continuous-time convolution
* Impulse and step responses
* Frequency response
* LTI system analysis

---

## Laboratory 3 – Signal Simulation and Fourier Analysis

This laboratory introduces signal simulation and frequency-domain analysis using Python. Students generate and combine sinusoidal signals, investigate the effects of signal parameters, and compute symbolic Fourier and inverse Fourier transforms using SymPy. The exercises provide a bridge between time-domain signal analysis and frequency-domain representation.

**Topics**

* Signal simulation
* Sinusoidal signal generation
* Signal superposition
* Fourier Transform
* Inverse Fourier Transform
* Symbolic computation with SymPy

## Software Requirements

* Python 3.x
* Google Colab or Jupyter Notebook
* NumPy
* SciPy
* Matplotlib
* SymPy

## Repository Structure

* **Student Notebooks** – Self-guided laboratory exercises with incomplete code for students to complete.
* **Instructor Notebooks** – Fully worked solutions with explanations and reference implementations.

These laboratories support active learning by combining mathematical theory with computational experimentation, enabling students to develop practical programming skills alongside a deeper understanding of signal processing concepts.
